# QuickShop Prototype

Fast static prototype for an online store with:

- Product categories (Men, Women, Children)
- Search and category filters
- Admin modal to add products quickly
- Local persistence via `localStorage`

## Run locally

Open `index.html` in any browser.

## Prototype notes

- This is a front-end prototype (no real backend, auth, or payments).
- Added products are saved in your browser only.
